## 安装步骤  
1 cp .env.example .env  
2 composer install  
3 php artisan key:generate  
4 php artisan admin:install  

* 注册
* 登陆
* 重置密码
* 通用api返回
* 短信验证
* 文档说明 http(s)://host/doc
* 管理后台 http(s)://host/admin 用户名:admin 密码:admin
